Featured Case Studies

High-Velocity IoT to ERP Data Integration

Lead Systems Integrator / Data Architect

Challenge: An external ERP vendor deployed a rigid, positional-indexing parser that failed to accurately record production data from a KADAR corrugator control machine during high-speed, dynamic manufacturing exceptions.

Solution: Conducted Root Cause Analysis (RCA) and spearheaded the architectural redesign from a fixed positional sequence to a dynamic, Data-Key Mapping logic. Enforced strict UAT deployment protocols to prevent vendor "testing in production."

Result: Achieved 100% data integrity between physical factory output and financial ERP ledgers, protecting the company's software investment and eliminating manual data entry for floor operators.

Enterprise Virtualization & Disaster Recovery

Infrastructure Architect

Challenge: Ensuring zero-downtime and data safety for a 24/7 large-scale manufacturing facility.

Solution: Architected a highly available server environment utilizing VMware ESXi and vSphere. Implemented automated, bare-metal, and file-level backups using Linux-based BackupPC with scheduled cron-job automation.

Result: Maintained 99.9% uptime for mission-critical ERP and Active Directory servers, completely insulating the business from data loss and hardware failures.

AI-Driven IoT Emotion Detection Pipeline

Machine Learning & IoT Integrator

Challenge: Processing unstructured analog data (human voice) in real-time to trigger physical hardware responses based on emotional context.

The Python script pipes the ML inference through the Tuya SDK/API to dynamically change the color of an RGB smart bulb.
